# repaso_progra
Repo para repasar programacion durante el verano

# Instrucciones 

* git add . /para agregar cosas que quiero commitear
* git commit -m'mensaje' / Para guardar localmente los archivos que agregue con add
* git push / Para subir todos los commits que no he subido

* Si estuviese trabajando alguien más hay que hacer 'git pull' para descargar los archivos
